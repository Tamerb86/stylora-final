# Dashboard Design Analysis

## Key Features from Reference Images

### 1. Impersonation Mode Banner (Orange/Red Gradient)
- Top banner with warning icon
- Text: "Du er logget inn som en annen salong (Impersonation Mode)"
- Subtext: "Alle handlinger utføres på vegne av denne salongen"
- "Avslutt innlogging" button on the right

### 2. Email Verification Banner (Yellow/Beige)
- Warning icon on left
- Title: "Bekreft e-postadressen din"
- Subtext: "Vi har sendt en bekreftelse til [email]"
- "Send på nytt" button
- Close (X) button

### 3. Main Dashboard Layout
- Title: "Dashboard" (with gradient colors - blue to purple/orange)
- Subtitle: "Oversikt over dagens aktivitet"
- "Ny avtale" button with gradient (blue to orange)

### 4. Welcome Card - "Velkommen til Stylora"
- Wave emoji icon
- Title: "Velkommen til Stylora"
- Description: "Kom i gang med å administrere salongen din"
- Three action buttons (purple gradient):
  - "Opprett ny avtale" (Calendar icon) - "Administrer timeboken din"
  - "Legg til nye kunder" (Users icon) - "Bygg kunderegisteret ditt"
  - "Administrer tjenester" (Trending icon) - "Sett opp priser og varighet"

### 5. Performance Card - "Dagens ytelse"
- Chart icon
- Title: "Dagens ytelse"
- Subtitle: "Sammendrag av aktivitet"
- Metrics:
  - "Fullførte avtaler": 0 (blue background)
  - "Omsetning": 0.00 NOK (green background)
  - "Ventende": 0 (orange/red background)
- "Se fullstendig analyse" link with arrow

### 6. Appointments Section
- "Dagens avtaler" title
- Calendar icon button
- "Totalt i dag" label
- Error indicator: "16 errors" (red pill)

## Design Elements
- Card-based layout with white backgrounds
- Soft shadows on cards
- Purple/blue gradient buttons
- Color-coded metrics (blue, green, orange/red)
- Clean spacing and typography
- Norwegian language throughout
